# Интеграции

Проект содержит набор feature-модулей в `app/features/*`:

- albumentations_integration
- comet_integration
- dvc_integration
- hyperparameter_tuning
- kfold_integration
- model_export
- model_validation
- sagemaker_integration
- sahi_integration
- и др.

## Конфигурация
- Основной файл интеграций: `integrations_config.json`.
- Есть миграции схемы/версии конфигурации через соответствующие use-case/репозитории.
