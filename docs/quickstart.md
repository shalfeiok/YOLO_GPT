# Быстрый старт (5 шагов)

1. Установите зависимости и запустите приложение (`python main.py`).
2. Перейдите на вкладку **«Датасеты»** и подготовьте/проверьте структуру датасета с `data.yaml`.
3. Перейдите на вкладку **«Обучение»**:
   - выберите модель (`yolo11n.pt` и т.д.),
   - задайте датасет,
   - укажите epochs/batch/imgsz/device,
   - нажмите старт обучения.
4. После первого прогона откройте **«Советник по обучению»**, выполните анализ и при необходимости нажмите **Apply** (рекомендации применяются в settings store, есть undo).
5. Для инференса используйте вкладку **«Детекция»**, для оценки качества — **«Валидация»**, для контроля выполнения — **«Задачи»**.

## Быстрые проверки

- Тесты: `pytest -q`
- Проверка стиля: `ruff check .`
