# Интеграция Comet ML

Официальный гайд: [Ultralytics — Comet](https://docs.ultralytics.com/ru/integrations/comet/).

## Назначение

Comet ML — платформа для сравнения экспериментов, визуализации метрик и управления моделями. При обучении YOLO автоматически логирует mAP, loss, гиперпараметры, матрицу ошибок и примеры предсказаний.

## Использование в приложении

1. Вкладка **«Интеграции и мониторинг»** → секция **B. Трекинг экспериментов (Comet ML)**.
2. Включите «Логировать эксперименты в Comet ML».
3. Укажите **Comet API Key** (из личного кабинета Comet).
4. Задайте **Project Name** (по умолчанию `yolo26-project`).
5. В «Дополнительные настройки» при необходимости измените:
   - COMET_MAX_IMAGE_PREDICTIONS
   - COMET_EVAL_BATCH_LOGGING_INTERVAL
   - COMET_EVAL_LOG_CONFUSION_MATRIX
   - COMET_MODE (online / offline / disabled).
6. При запуске обучения приложение выставляет переменные окружения `COMET_*`; Ultralytics при установленном `comet-ml` автоматически создаёт эксперимент и логирует метрики.

## Установка

```bash
pip install comet-ml ultralytics
```

## Открытие панели Comet

Кнопка «Открыть панель Comet в браузере» открывает сайт Comet. После обучения зайдите в свой проект, чтобы просмотреть эксперименты и графики.
