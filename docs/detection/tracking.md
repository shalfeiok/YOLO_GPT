# Детекция и трекинг

В проекте присутствуют отдельные view для tracking/segmentation/pose/classification.

## Текущее состояние
- Базовая инфраструктура вкладок и lazy-loading реализована.
- Часть экранов может быть упрощённой/placeholder в текущей итерации.

## Рекомендация
Используйте эти вкладки как расширение базового сценария «Training → Detection → Validation».
