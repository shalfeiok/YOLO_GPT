# Очереди, job runner и логи

## Компоненты
- `EventBus` — доставка событий.
- `JobRunner` — выполнение фоновых задач в процессе.
- `ProcessJobRunner` — выполнение в отдельном процессе.
- `JobRegistry` — агрегирует состояние задач.
- `JsonlJobEventStore` — персистентность истории задач.

## Что видно пользователю
Во вкладке «Задачи» можно:
- фильтровать и искать,
- смотреть логи,
- отменять/повторять,
- открывать run artifacts.
