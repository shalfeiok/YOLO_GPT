# Интеграция Albumentations

Официальный гайд: [Ultralytics — Albumentations](https://docs.ultralytics.com/ru/integrations/albumentations/).

## Назначение

Albumentations — быстрая библиотека аугментации изображений (70+ преобразований). При обучении YOLO26 улучшает обобщающую способность модели за счёт геометрических и цветовых преобразований.

## Использование в приложении

1. Вкладка **«Интеграции и мониторинг»** → секция **A. Аугментация данных (Albumentations)**.
2. Включите «Включить Albumentations во время обучения».
3. Выберите режим:
   - **Стандартные трансформы** — Ultralytics применяет встроенный набор при установленном пакете `albumentations`.
   - **Кастомные трансформы** — нажмите «Редактировать кастомные трансформы», добавьте трансформы (Blur, MedianBlur, ToGray, CLAHE, GaussNoise и др.) и задайте вероятность `p`.
4. Вероятность (p) — общая вероятность применения набора (слайдер 0–1).
5. При следующем запуске обучения из вкладки «Обучение» кастомные трансформы передаются в `model.train(augmentations=...)`.

## Установка

```bash
pip install albumentations ultralytics
```

Если пакет не установлен, аугментации Albumentations при обучении не применяются.

## Параметры трансформов (кратко)

| Трансформ | Описание | Параметры в интеграции |
|-----------|----------|------------------------|
| Blur | Размытие по ядру | blur_limit, p |
| MedianBlur | Медианное размытие | blur_limit, p |
| ToGray | В оттенки серого | p |
| CLAHE | Контраст (CLAHE) | clip_limit, p |
| GaussNoise | Гауссов шум | var_limit, p |
| RandomBrightnessContrast | Яркость/контраст | p |
| HueSaturationValue | HSV | p |

Подробности: [документация Albumentations](https://albumentations.ai/docs/).
