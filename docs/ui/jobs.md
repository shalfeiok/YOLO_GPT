# Вкладка «Задачи»

## Что это
Мониторинг фоновых задач (`JobRunner`, `ProcessJobRunner`, `JobRegistry`).

## Функции
- Таблица задач и статусов.
- Фильтры.
- Просмотр/поиск логов.
- Cancel/Retry.
- Быстрый доступ к run folder, манифестам, весам, графикам.

## Практика
Используйте вкладку для расследования падений и анализа ретраев/таймаутов.
