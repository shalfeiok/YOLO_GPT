# Вкладка «Датасеты»

## Что это
Экран для работы с датасетами: подготовка, базовая проверка, запуск вспомогательных операций.

## Для чего
- Проверить, что датасет соответствует формату YOLO.
- Убедиться, что есть `train/val` и корректный `data.yaml`.

## Где находится
Sidebar → **Datasets**.

## Параметры и элементы (типовые)

| Элемент | Смысл | Рекомендации | Риски |
|---|---|---|---|
| Path к датасету | Корневая папка или data.yaml | Использовать абсолютный путь | Неверный путь ломает train/val |
| Проверка структуры | Валидация train/val/labels | Проверять перед каждым обучением | Тихие ошибки разметки |

## Частые ошибки
- Нет `data.yaml`.
- Не найден `val/images`.
- Несоответствие путей между Windows/Linux.
