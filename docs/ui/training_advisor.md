# Вкладка «Советник по обучению»

## Что это
Модуль, который объединяет:
- инспекцию датасета,
- чтение артефактов прошлого запуска,
- быструю оценку модели,
- генерацию рекомендаций для TrainingConfig.

## Для чего
Ускорить подбор параметров без ручного анализа логов.

## Где
Sidebar → **Training Advisor**.

## Режимы
- Quick
- Deep (добавляет предупреждение о heuristic calibration, без тяжёлых вычислений).

## Apply/Undo
- **Apply**: рекомендации применяются в `AppSettingsStore` как новый training snapshot.
- **Undo**: откат к предыдущему snapshot.

## Примеры рекомендаций
- При imbalance: увеличить mixup/mosaic.
- При underfitting: увеличить epochs, снизить lr0.
- При OOM в прошлых предупреждениях: уменьшить batch/imgsz.
