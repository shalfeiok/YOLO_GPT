# Вкладка «Интеграции»

## Что это
Единая точка включения/настройки интеграций (Comet, DVC, Albumentations и др. модули в `app/features/*`).

## Где
Sidebar → **Integrations**.

## Для чего
- Экспорт/импорт конфигурации интеграций.
- Централизованно хранить параметры внешних инструментов.

## Важно
Интеграции подключаются мягко: при сбое конфигурации обучение продолжается с default поведением.
