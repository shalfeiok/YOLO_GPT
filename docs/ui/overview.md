# UI: обзор вкладок

В приложении используется `QMainWindow` + sidebar + lazy-loaded stack. Основные вкладки:

- Датасеты
- Обучение
- Советник по обучению
- Детекция
- Валидация
- Сегментация
- Pose
- Классификация
- Трекинг
- Автоаннотирование
- Бенчмарк
- Эксперименты
- Интеграции
- Задачи

## Навигация

- `Ctrl+1..9` — быстрые вкладки.
- `Ctrl+K` — командная палитра.

## Важно

Часть вкладок может быть в состоянии placeholder/минимального функционала в текущей фазе разработки — это нормальное поведение.
