# Вкладка «Детекция»

## Что это
Запуск инференса модели (PyTorch/ONNX через адаптеры) на выбранном источнике кадров.

## Где
Sidebar → **Detection**.

## Параметры

| Параметр | Смысл | Рекомендации | Риски |
|---|---|---|---|
| model_path | Путь к весам | использовать проверенные веса | неверный путь/формат |
| confidence | Порог confidence | 0.25–0.5 | шум или пропуски |
| iou_threshold | NMS IoU | 0.45 как старт | дубли боксов/агрессивный NMS |

## Сценарий
1. Выбрать модель.
2. Указать источник.
3. Нажать Start.
4. Остановить через Stop (best-effort cleanup use-case).
