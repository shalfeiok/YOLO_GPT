# Расширенные параметры обучения

## Что это
Поля из `advanced_options` (`ADVANCED_DEFAULTS`) в `TrainingConfig`.

## Ключевые параметры

| Параметр | Назначение | Когда менять |
|---|---|---|
| lr0 / lrf | Стартовый/финальный lr | нестабильная сходимость |
| mosaic / mixup | Аугментации | imbalance, регуляризация |
| close_mosaic | Выключение mosaic к концу | борьба с overfitting |
| amp | mixed precision | ускорение на поддерживаемом GPU |
| weight_decay | регуляризация | переобучение |

## Рекомендации
- Меняйте 1–2 параметра за итерацию.
- Сверяйтесь с отчетом советника и метриками runs.
