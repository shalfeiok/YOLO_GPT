# Интеграция DVC / DVCLive

Официальный гайд: [Ultralytics — DVC](https://docs.ultralytics.com/ru/integrations/dvc/).

## Назначение

DVCLive + DVC обеспечивают версионирование экспериментов, интеграцию с Git и генерацию сравнительных отчётов. Позволяет отследить, какая версия кода и данных дала лучший результат.

## Использование в приложении

1. Вкладка **«Интеграции и мониторинг»** → секция **C. Версионирование экспериментов (DVCLive)**.
2. Включите «Отслеживать эксперименты с DVC».
3. Убедитесь, что в проекте инициализирован Git (`git init`). Индикаторы показывают «Git репозиторий инициализирован: Да/Нет» и «DVC инициализирован: Да/Нет».
4. Нажмите **«Инициализировать DVC в проекте»** — выполнится `dvc init` в корне проекта.
5. После нескольких запусков обучения нажмите **«Сгенерировать отчёт DVC plots»** — выполнится `dvc plots diff`; откроется HTML с графиками сравнения экспериментов.

## Установка

```bash
pip install ultralytics dvclive
# или
pip install dvc
```

При установленном DVCLive Ultralytics автоматически логирует метрики в DVC при обучении (если проект — DVC-репозиторий).
