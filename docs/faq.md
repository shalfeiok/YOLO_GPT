# FAQ

## Можно ли запускать без GPU?
Да, на CPU. Медленнее, но поддерживается.

## Где взять best.pt?
В `runs/train/<exp>/weights/best.pt`.

## Почему советник меняет epochs/batch/imgsz?
Он использует heuristic-анализ датасета, прошлых артефактов и метрик.

## Есть ли полный e2e UI тест?
В проекте используется smoke-подход для UI (инициализация виджетов), чтобы тесты были стабильными в CI.
