# Валидация датасета

`DatasetInspector` проверяет:

- наличие `data.yaml`,
- наличие images/labels по split train/val/test,
- битые изображения,
- пропущенные label-файлы,
- пустые label-файлы,
- ошибки формата label,
- bbox out-of-range,
- дисбаланс классов (>5x),
- выход class id за пределы `names[]`.

## Что делать при ошибках
- Исправить разметку (особенно координаты и class id).
- Убедиться, что names в data.yaml соответствует реальным id.
- Пересобрать train/val при сильном дисбалансе.
